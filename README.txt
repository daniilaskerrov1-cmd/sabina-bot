Railway deploy package for Sabina Gift Bot

STEPS:

1) Open https://railway.app
2) Login with Google or GitHub
3) Click "New Project"
4) Click "Empty Project"
5) Click "Upload Files"
6) Upload ALL files from this archive
7) Wait for Deploy
8) Bot works 24/7

IMPORTANT:
Insert your bot token in bot.py before uploading.

Find line:

TOKEN = os.getenv("BOT_TOKEN", "PASTE_YOUR_TOKEN_HERE")

Replace with your real token.
